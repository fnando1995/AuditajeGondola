# Supermarket Shelf Audit Project

Este repositorio contiene el código desarrollado como parte de mi proyecto de tesis titulado "Auditaje Automático de Góndolas en Supermercados Usando Técnicas de Aprendizaje Profundo y Visión por Computador". El proyecto aborda problemas encontrados en los supermercados unidos en una sola macro-tarea denominada: Auditaje de Góndolas.


## Contenido del Repositorio
- `requirements.txt`: Archivo de los requerimientos en python.
- `codes/`: Carpeta con los archivos para ejecutar paso a paso las secuencias presentes en el proyecto.
